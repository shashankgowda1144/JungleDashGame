# JungleDashGame

It is an exciting platformer where the player controls a character navigating through a jungle environment. The goal is to dash through various levels, avoid obstacles, collect items, and complete challenges.The game features multiple levels, each with increasing difficulty, and power-ups to assist players in their journey. The goal is to finish each level quickly and efficiently while earning the highest score.

Features:

- Level-based gameplay with unique obstacles
- Collectible power-ups for added advantages
- Challenging platforming mechanics

How to Play:

- Start the game by pressing the Start button.
- Use the controls to navigate your character and avoid obstacles.
- Collect items to gain extra points and power-ups.
- Try to last as long as possible while earning the highest score!
